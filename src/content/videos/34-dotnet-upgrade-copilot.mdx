## Overview
Complete workflow for upgrading legacy .NET apps with GitHub Copilot:
- Assessing a legacy .NET app for modernization
- Planning an upgrade with GitHub Copilot
- Executing the migration to modern .NET
- Using Copilot to speed up and de-risk .NET upgrades
- Handling unknown dependencies and breaking changes

Ideal for developers working with older .NET Framework or early .NET Core apps who want an AI-assisted upgrade path.
